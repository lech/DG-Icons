
# DG icons v1

--------------------------------------------------------------------------------

VMT Files are set for DG_red and DG_blu respectively.

## Format info:

Format: `BGRA8888` *_uncrushed.vtf large files

Format: `DTX5` *_crushed.vtf itsy bitsy files

### Shared File attributes:
```
Mipmap Filter: Box
Sharpen Filter: Sharpen Soft
Mipmaps: True
Thumbnail: True
Clamp S
Clamp T
No Level of Detail
```
